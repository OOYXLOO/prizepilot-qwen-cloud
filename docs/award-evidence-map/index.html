What Is Not Claimed

Honest boundary: PrizePilot now claims one completed live Qwen/DashScope smoke proof captured with a runtime-only key. It does not claim a live Alibaba Cloud endpoint until the account owner approves deployment and verifies a public endpoint.
  • No API keys, raw headers, cookies, billing identifiers, tax/KYC data, or payout information are published.
  • No private console screenshots are required for judges to review the submitted public package.
  • If live proof is added later, Devpost should be updated only after the evidence is actually reachable.
