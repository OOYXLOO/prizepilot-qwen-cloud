# Novus Integration Checklist

Purpose: satisfy the Mind the Product requirement without overstating proof.

## Current Status

- Devpost registration: complete
- Novus signup: started
- Novus email verification: required
- Novus password creation: required on the official email link
- Novus installed/connected: not yet

## After Email Verification

1. Sign in at `https://novus.pendo.io/`.
2. Follow the official Novus onboarding path for connecting or installing Novus.
3. If Novus asks for GitHub/OAuth, use the public `OOYXLOO/prizepilot-qwen-cloud` repository only after confirming the requested permissions are limited to this goal.
4. If Novus provides install code, add it through a non-secret configuration path. Do not commit private tokens, client secrets, API keys, or account-specific write credentials.
5. Run the local web app and confirm `/api/novus-readiness` is updated only after installation evidence exists.
6. Capture the public proof needed for Devpost: deployed URL, Novus installed/connected status, and demo video URL.
