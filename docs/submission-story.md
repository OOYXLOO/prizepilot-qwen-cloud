# Submission Story

PrizePilot is an Autopilot Agent for turning hackathon and bounty pages into accountable execution plans. It ranks prize routes, chooses realistic targets, prepares project/blog/Devpost drafts, and creates a human action queue for sensitive steps.

The demo focuses on a real portfolio: Splunk feedback first for a USD 200 near-term prize, Qwen Blog Post Award second for a USD 500 multi-winner prize, and higher-friction bounties as backups.

The project demonstrates why agent autonomy needs boundaries. PrizePilot can prepare artifacts and run local checks, but it requires user approval for accounts, API keys, public uploads, posts, cloud deployment, and final submissions.
